# Occidata guide deprecated

This repository no longer targets Occidata as the execution environment.

Use Grid'5000 instead:

- main guide: `docs/grid5000.md`
- setup helper: `scripts/grid5000_setup.sh`

The old `scripts/occidata_setup.sh` entrypoint is kept only as a compatibility wrapper and now delegates to the Grid'5000 setup script.
